# productivity-app
A colorful Productivity Tracker web application built using Flask and MongoDB that allows users to add, update, delete, and manage daily tasks efficiently.
A colorful and modern task management web application built using Flask, MongoDB, Python, HTML, and CSS.

---

## 📌 Project Overview

This Productivity Tracker helps users manage their daily tasks efficiently. Users can add new tasks, update task priority, toggle task status (Pending/Completed), and delete tasks. The app features a clean UI with colorful styling for better user experience.

---
## 🛠️ Technologies Used

- Python
- Flask
- MongoDB
- PyMongo
- HTML5
- CSS3

---

## ✨ Features

- ➕ Add New Tasks
- 🗑️ Delete Tasks
- 🔄 Toggle Task Status (Pending / Completed)
- 🔺 Update Task Priority (High / Medium / Low)
- 🎨 Colorful and Responsive UI
- 🗂 MongoDB Database Integration

## ▶️ How to Run

1. Install dependencies:
   pip install flask pymongo

2. Start MongoDB server

3. Run the app:
   python app.py

4. Open browser:
   http://127.0.0.1:5000
